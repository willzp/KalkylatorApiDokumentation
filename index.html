GET Kalkylator Api

----------------------------

Get kalkylator api är ett enkelt api som kan räkna på tre olika räknesätt

• Plus

• Minus

• Gånger

----------------------------

Nödvändiga Parametrar

• nr1

• nr2

• typ

----------------------------

Exempel URL

http://localhost:8080/kalkylator?nr1=500&nr2=500&typ=plus

Du får tillbaka svaret: Summan av 500 + 500 = 1000
